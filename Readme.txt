# AllPilar Website

## Description
AllPilar is a responsive website designed to showcase services, provide forms, and deliver detailed service information. This project includes several web pages such as a home page, starter page, service details, and forms.

## Project Structure
- **index.html**: The main homepage of the website.
- **sarter-page.html**: The starter page for specific content.
- **service-details.html**: Detailed information about services.
- **forms/**: Contains form-related files and assets.
- **assets/**: Includes styles, images, and other resources.

## Features
- Mobile-responsive design
- Easy navigation
- Structured content layout
